Project information

I made this project in the academic year of 2023 for the course Systems Engineering & Analysis. I had to do this together with 2 other students. The assignment for the project came from a fictitious client who wanted to make mathematics more interesting in technical subjects.

For more information: GitHub Repo

Soccer Robot

The idea we came up with was to make robots that could play soccer against each other. And this allowed us to explain the math involved in these little robots. During this project I mainly worked on the electronic circuit and I also wrote the code for the Arduino together with a fellow student.

We made three different versions of the robot, the first was that the robot is controllable by a wired joystick, the second is that the robot is controlled by a mobile phone that connects to the robot via bluetooth, and the third is that the robot is wireless. controlled by means of radio frequency communication

The circuit of the robot contains the following parts:

  • Arduino Uno
  • L293D Motor Driver
  • 2 x DC Motor
  • Joystick
  • Arduino Nano
  • nRF24L01
  • HC-06 Bluetooth module
