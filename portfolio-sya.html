Project information

This robot was a 2023 Systems Engineering & Analysis project with 2 fellow students. The assignment for the project came from a fictitious client who wanted to make mathematics more interesting in technical classes.

For more information: GitHub Repo

Soccer Robot

We came up with the idea to make robots that could play soccer against each other. This allowed us to explain the mathematics involved in these little robots. During this project I mainly worked on the electronic circuit and wrote the code for the Arduino together with a fellow student.

We made three different versions of the robot, the first robot was controllable by a wired joystick, the second was controlled by a mobile phone that connected to the robot via bluetooth, and the third was controlled by means of a wireless RF connection.

The circuit of the robot contained the following parts:

  • Arduino Uno
  • L293D Motor Driver
  • 2 x DC Motor
  • Joystick
  • Arduino Nano
  • nRF24L01
  • HC-06 Bluetooth module
